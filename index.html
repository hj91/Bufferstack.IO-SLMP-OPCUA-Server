  • Network interface card

Supported PLCs

  • iQ-F series controllers

  • L series PLCs

Protocol Support

Installation Procedure

  1. Download installer package from link above
  2. Execute installer with administrator privileges
  3. Accept End-User License Agreement (EULA)
  4. Select installation directory (default: C:\Program Files)
  5. Complete installation wizard process
  6. Launch application from Start Menu shortcut
  7. Configure PLC connections via web interface (port 8080)
  8. Verify OPC UA server connection (default port 4840)
